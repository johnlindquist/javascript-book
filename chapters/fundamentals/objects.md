# Objects

An `Object` groups together related data using `{}`:
```js
const person = {firstName: "Kim", lastName: "Wilkes"}
```

An `Object` can get quite large and span across multiple lines:
```js
const person = {
  firstName: "Kim",
  lastName: "Wilkes",
  planet: "Earth",
  isAwesome: true,
  parts: ["Head", "Shoulders", "Knees", "Toes"]
}
```
